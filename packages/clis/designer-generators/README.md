### Scripts

- **$** `npm run test` / `npm run test:watch` - uses `mocha` + `ts-node/register`.
- **$** `npm run lint` / `npm run lint:fix` - uses [@noodlestan/eslint-config](https://www.npmjs.com/package/@noodlestan/eslint-config)
- **$** `npm run dev` - uses `tsc --watch`
