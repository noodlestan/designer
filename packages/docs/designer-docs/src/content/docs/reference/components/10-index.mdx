## Common props

Some components accept a `d` prop.

This prop serves to identify the decision via its `id` or `name` attributes.

<Code code={`<ShowDecisionCard d="Green Foreground" />`} title={'page.mdx'} lang="astro" />

<ShowDecisionCard d="Green Foreground" />

You can also provide a `DecisionInput` data object, or even a partial one,
as long the `name` attribute is present.

<Code
    code={`<ShowDecisionCard d={{name: "Green Foreground"}} />`}
    title={'page.mdx'}
    lang="astro"
/>

    <ShowDecisionCard d="Green Foreground" />
