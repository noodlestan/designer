```ts
import { ShowDecisionCard } from '@noodlestan/designer-shows/astro';
```

Renders the appropriate visualization according to the decision type, along with any
decision details you might want to display.

<FlexStack vpad>
    <ShowDecisionCard store={store} d="Action Color" />
</FlexStack>

## Props

### ༶ [DecisionProps](/components/Astro/Props): `store` and `d`

These two **mandatory** props serve to identify and retrieve the decision.

### ༶ [ValueProps](/components/Astro/Props#valueprops): `value` and `viz`

You can use these props to control how the visualization and value are displayed.

In this example we are rendering a [ColorValueDecision](/decision-models/color-value)
and both

-   hiding the visualization;
-   and displaying its value in `rgb` and `hsl` formats.

```mdx
<ShowDecisionCard store={store} d="Action Color" value={['rgb', 'hsl']} />
```

<FlexStack vpad>
    <ShowDecisionCard store={store} d="Action Color" value={['rgb', 'hsl']} />
</FlexStack>

### ༶ `name` (true): boolean

Provide this flag to hide the decision's name.

```mdx
<ShowDecisionCard store={store} d="Space Unit" name={false} />
```

<FlexStack vpad>
    <ShowDecisionCard store={store} d="Space Unit" name={false} />
</FlexStack>

### ༶ `description` (true): boolean

Provide this flag to hide the decision's description.

```mdx
<ShowDecisionCard store={store} d="Action Color" description={false} />
```

<ShowDecisionCard store={store} d="Action Color" description={false} />
