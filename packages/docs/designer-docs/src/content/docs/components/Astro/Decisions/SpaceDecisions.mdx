## Components

These components render space related decisions.

```ts
import { ... } from '@noodlestan/designer-shows/astro';
```

### ShowSpaceValueDecision

Renders a [SpaceValueDecision](/decision-models/space-value).

Use the [viz prop](#-viz-true-boolean--spacevizname) to chose the visualization type.

```mdx
<ShowSpaceValueDecision store={store} d="Space Unit" size="s" />
```

<CardGrid>
    <ShowSpaceValueDecision store={store} d="Space Unit" size="s" />
</CardGrid>

### ShowSpaceScaleDecision

Renders a [SpaceScaleDecision](/decision-models/space-scale).

Use the [viz prop](#-viz-true-boolean--spacevizname) to chose the visualization type.

```mdx
<ShowSpaceScaleDecision store={store} d="Sizing Space Scale" size="xs" />
```

<CardGrid>
    <ShowSpaceScaleDecision store={store} d="Sizing Space Scale" size="xs" />
</CardGrid>

## Props

### ༶ [DecisionProps](/components/Astro/Props): `store` and `d` (mandatory)

These two **mandatory** props serve to identify and retrieve the decision.

### ༶ [ValueProps](/components/Astro/Props#-value-true-boolean--string--object): `value` and `size`

These optional props control the size of the visualization an whether the value is shown.

### ༶ `viz` (true): boolean | [SpaceVizName](/api/designer-shows/Types/SpaceVizName)

Chose the visualization type.

Refer to [SpaceVizName](/api/designer-shows/Types/SpaceVizName) for available visualizations.
