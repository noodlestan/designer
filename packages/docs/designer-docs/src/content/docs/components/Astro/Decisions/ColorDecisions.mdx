## Props

### ༶ [DecisionProps](/components/Astro/Props): `store` and `d`

These two **mandatory** props serve to identify and retrieve the decision.

### ༶ [ValueProps ༶ value](/components/Astro/Props/#-value-true-boolean--string--object)

The `value` prop controls how the decision value is displayed.

#### Color Values

For decisions of type [ColorValueDecision](/decision-models/color-value) and
[ColorSetDecision](/decision-models/color-set), the `value` prop can be used
not only to show/hide the decision value but also to display
it in one or more color spaces.

Provide a `string` with one of [ColorSpaceName](/api/designer-decisions/Types/ColorSpaceName/)
to specify which color space / format to use.

```mdx
<ShowDecision d="Brand Blue" store={store} size="s" value="oklch" />
```

<FlexRow vpad>
    <ShowDecision d="Brand Pink" store={store} size="s" value={'oklch'} />
</FlexRow>

Provide an `array` of [ColorSpaceName](/api/designer-decisions/Types/ColorSpaceName/)
to list several color spaces in your preferred order.

```mdx
<ShowDecision d="Brand Pink" store={store} size="s" value={['oklch', 'rgb']} />
```

<FlexRow vpad>
    <ShowDecision d="Brand Pink" store={store} size="s" value={['oklch', 'rgb']} />
</FlexRow>

### ༶ [ValueProps ༶ size](/components/Astro/Props#-size-auto-showvaluesize)

The `size` optional prop controls the size of the visualization.

Refer to [ShowValueSize](/api/designer-shows/Types/ShowValueSize) for available sizes.

### ༶ [ValueProps ༶ viz](/components/Astro/Props/#-viz-true-boolean--string)

Chose visualization type.

In this example we set the visualization to `bg`.

```mdx
<ShowDecision store={store} d="Action Color" size="m" viz="bg" options={{ contrast: '#FDFBFC' }} />
```

<CardGrid>
    <ShowDecision
        store={store}
        d="Action Color"
        size="m"
        viz="bg"
        options={{ contrast: '#FDFBFC' }}
    />
</CardGrid>

Refer to [ColorVizName](/api/designer-shows/Types/ColorVizName) for available visualizations.

### ༶ [ValueProps ༶ options](/components/Astro/Props/#-options-optional-object)

Configure the visualization.

In this example we change it to `fg` and provide the
additional `options` prop for defining the visualization options.

```mdx
<ShowDecision store={store} d="Action Color" size="m" viz="fg" options={{ contrast: '#FDFBFC' }} />
```

<CardGrid>
    <ShowDecision
        store={store}
        d="Action Color"
        size="m"
        viz="fg"
        options={{ mode: 'text', contrast: '#FDFBFC' }}
    />
</CardGrid>

Refer to [ColorVizOptions](/api/designer-shows/Types/ColorVizOptions) for available options.
