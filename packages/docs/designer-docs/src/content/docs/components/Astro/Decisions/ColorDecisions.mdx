import { decisionLoader } from '../../../../../decisions';

export const store = await decisionLoader();

## Props

##### ༶ `store` and `d` (mandatory)

These components require the two mandatory [DecisionProps](/components/Astro/Props) - `store`
and `d` - to identify and retrieve the decision.

##### ༶ `value`, `viz`, and `size` (optional)

They also accepts [ValueProps](/components/Astro/Props#valueprops) - `value`,
`viz` and `size` - to control how the visualization and value are displayed.

## Components

```ts
import { ... } from '@noodlestan/designer-shows/astro';
```
