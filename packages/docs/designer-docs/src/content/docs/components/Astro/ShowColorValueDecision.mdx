
<CardGrid>
    <ShowColorValueDecision d="Green Foreground" store={store} />
</CardGrid>

## Props

##### ༶ `store` and `d` (mandatory)

This component requires the two mandatory [DecisionProps](/components/astro/props) - `store`
and `d` - to identify and retrieve the decision.

##### ༶ `value`, `viz`, and `size` (optional)

It also accepts [ValueProps](/components/astro/props#valueprops) - `value`,
`viz` and `size` - to control how the visualization and value are displayed.

```mdx
<ShowColorValueDecision store={store} d="Green Foreground" size="m" value={['rgb', 'hsl']} />
```
