Available visualizations for [Space Decisions](/components/Astro/Decisions/space)

```ts
type SpaceVizName = 'bar-h' | 'bar-v' | 'square' | 'circle';
```

When rendering decisions you can chose a different visualization by
setting the `viz` [prop](/components/Astro/Props#valueprops) to one of these.

```mdx
<ShowSpaceValueDecision store={store} d="Space Unit" size="xs" viz="bar-h" />
```

#### `viz="bar-h"` (default)

<FlexStack vpad>
    <ShowDecision store={store} d="Space Unit" size="xs" viz="bar-h" />
</FlexStack>
