Available visualizations for [Space Values](/components/Astro/Composable/Values/#showspacevalue)

```ts
type SpaceVizName = 'bar-h' | 'bar-v' | 'square' | 'circle';
```

When rendering decisions you can chose a different visualization by
setting the `viz` [prop](/components/Astro/Props#-viz-true-boolean--string) to one of these.

```mdx
<ShowSpaceValueDecision store={store} d="Space Unit" viz="..." />
```
