import { ShowDecision } from '@noodlestan/designer-shows/astro';

Available visualizations for [Color Values](/integrations/Astro/Composable/Values/#showcolorvalue).

```ts
export type ColorVizName = 'swatch' | 'fg' | 'bg';
```

When rendering color decisions you can chose a different visualization by
setting the [viz prop](/integrations/Astro/Props#-viz-true-boolean--string) to one of these.

```mdx
<ShowColorValueDecision d="Brand Black" viz="..." options={options} />
```

## Type Values

### `swatch` (default)

Flat color surface.

No options.

```mdx
<ShowDecision d="Brand Pink" size="m" viz="swatch" />
```

<ShowDecision d="Brand Pink" size="m" viz="swatch" />

### `fg`

Render the decision value as the foreground content color.

```mdx
<ShowDecision d="Brand Pink" size="m" viz="fg" options={{ contrast: 'oklch(24% 0.08 257)' }} />
```

<ShowDecision d="Brand Pink" size="m" viz="fg" options={{ contrast: 'oklch(24% 0.08 257)' }} />

Refer to [ColorVizOptions](/api/designer-shows/Types/ColorVizOptions/) for details on options.

### `bg`

Render the decision value as the background with custom element(s) on top.

```mdx
<ShowDecision
    d="Action Color"
    size="m"
    viz="bg"
    options={{ mode: 'text', contrast: 'oklch(99% 0.002 345)' }}
/>
```

<ShowDecision
    d="Action Color"
    size="m"
    viz="bg"
    options={{ mode: 'text', contrast: 'oklch(99% 0.002 345)' }}
/>

Refer to [ColorVizOptions](/api/designer-shows/Types/ColorVizOptions/) for details on options.
