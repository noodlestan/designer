Captures an error produced during [buildStaticStore()](/api/designer-functions/Builder/buildStaticStore).

```ts
export type BuilderError = {
    name: string;
    error?: unknown;
    message: () => string;
};
```

## Errors

### BuilderUnexpectedError

An unexpected error occurred while building the store.

### BuilderOptionsError

The [BuilderOptions](/api/designer-functions/Builder/Types/BuilderOptions), either provided to `createBuilderContext()` or expressed in the `dd.config.mjs` [configuration file](/api/configuration), are invalid.

### BuilderSourceError

One of the [DecisionSource](/api/designer-decisions/Meta/Types/DecisionSource) or [SchemaSource](/api/designer-decisions/Meta/Types/SchemaSource), provided to `createBuilderContext()` or expressed in the `dd.config.mjs` [configuration file](/api/configuration), is invalid.
