Provides an API to validate a decision [InputRecord](/api/designer-decisions/Inputs/Types/InputRecord).

```ts
export type DecisionValidator = {
    schemas: () => SchemaData[];
    validate: (decision: InputRecord) => ErrorObject[] | null;
};
```

<Aside type="tip" title="Advanced API">
    Access to this API is only required in advanced use cases. In most scenarios you should be using
    [staticStoreBuilder()](api/designer-functions/Builder/staticStoreBuilder).
</Aside>
