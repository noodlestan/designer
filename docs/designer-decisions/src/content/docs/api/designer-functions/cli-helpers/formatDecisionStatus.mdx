type formatDecisionStatus = (

Given a produced decision status, as returned by
[produceDecisionStatus()](/api/designer-decisions/cli-helpers/produceDecisionStatus)
returns a string with the decision status, value, details, and errors.

## Return value

`string`

## Params

##### à¼¶ `status`: DecisionStatus

As returned by
[produceDecisionStatus()](/api/designer-decisions/cli-helpers/produceDecisionStatus).

##### à¼¶ `show`: `ShowOptions[]`

The following values are accepted:

-   `status`
-   `uuid`
-   `name`
-   `model`
-   `value`
-   `error-count`

how status and errors are displayed:

## Usage

```js
const store = await decisionLoader();
const produced = produceDecisions(store);
produced.decisions().forEach(status => console.info(formatDecisionStatus(status)));
```
