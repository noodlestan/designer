```ts
type produceDecisions = (store: StaticStore) => ProducedDecisionStore;
```

Given a populated [StaticStore](/api/designer-functions/Store/Types/StaticStore) produces all possible decision values and exposes all decision values, contexts and aggregated errors.

## Return Value

Returns a `ProducedDecisionStore` object.

```ts
export type ProducedDecisionStore = {
    decisions: () => ProducedDecisionStatus[];
    hasErrors: () => boolean;
    errors: {
        count: () => number;
        store: () => StaticStoreError[];
        validation: () => InputValidationError[];
        value: () => (DecisionError | DecisionValueError)[];
    };
    summary: () => string;
};
```

## Params

#### à¼¶ `store`: [StaticStore](/api/designer-functions/Store/Types/StaticStore)

&nbsp;

## Usage

```js
const store = await loader();
const produced = produceDecisions(store);
console.info(produced.summary());
```
