Creates a [StoreContext](/api/designer-functions/Store/Types/StoreContext) with an optional configuration.

```ts
type createStoreContext = (context: StoreContext, validatedMap: StaticValidatedMap) => Store;
```

## Return value

Returns a [StoreContext](/api/designer-functions/Store/Types/StoreContext), required to create both for [staticStoreBuilder()](/api/designer-functions/Builder/staticStoreBuilder) and [createStaticStore()](/api/designer-functions/Store/createStaticStore)

## Params

#### ༶ `context`: [StoreContext](/api/designer-functions/Store/Types/StoreContext)

A context object that exposes the [StoreOptions](/api/designer-functions/Store/Types/StoreOptions) and collects eventual errors produced building the store.

It is recommended to declare the options object directly in a `dd.config.mjs` file in the root of the project and then load it via [loadConfig()](/api/designer-functions/config/loadConfig).

## Params

#### ༶ `validatedMap`: [StaticValidatedMap](/api/designer-decisions/Inputs/Types/StaticValidatedMap)

Holds all decision input data and provides an API to retrieve decision records and validation errors.

&nbsp;

## Usage

Create a context with configuration and provide to the `staticStoreBuilder()`.

```ts
const config = await loadConfig();
const context = createStoreContext(config.store);
const builder = staticStoreBuilder(context);
```
