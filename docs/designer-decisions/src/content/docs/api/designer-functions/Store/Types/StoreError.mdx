Captures an error produced while building a decision [Store](/api/designer-functions/Store/Types/Store).

```ts
export type StoreError = {
    name: string;
    error?: unknown;
    message: () => string;
};
```

## Errors

### StoreUnexpectedError

An unexpected error occurred while building the store.

### StoreOptionsError

The [StoreOptions](/api/designer-functions/Store/Types/StoreOptions), either provided to `createStoreContext()` or expressed in the [configuration](/api/configuration), are invalid.

### StoreSourceError

One of the [DecisionSource](/api/designer-decisions/Meta/Types/DecisionSource) or [SchemaSource](/api/designer-decisions/Meta/Types/SchemaSource), provided to `createStoreContext()` or expressed in the [configuration](/api/configuration), is invalid.
