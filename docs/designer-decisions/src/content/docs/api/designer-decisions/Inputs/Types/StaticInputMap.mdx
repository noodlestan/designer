Holds all decision input data and provides an API to retrieve decision records and validation errors.

## Methods

#### ༶ `record(ref, contexts)`

**Return Value:** [InputRecord](/api/designer-decisions/Inputs/Types/InputRecord) or `undefined` if not matched.

**Parameters:**

-   `ref`: [DecisionRef](/api/designer-decisions/Decisions/Types/DecisionRef)
-   `contexts`: [LookupContexts](/api/designer-decisions/Decisions/Types/LookupContexts) - When a decision `ref` matches more than one decision input and no `contexts` were provided, it resolves to the decision with less (or higher) context.

#### ༶ `records(filter?)`

**Return Value:** [InputRecord](/api/designer-decisions/Inputs/Types/InputRecord)[]

**Parameters:**

-   `filter`: (optional) A filter function `(item: InputRecord) => boolean`

#### ༶ `hasErrors()`

**Return Value:** `boolean` indicating if there were validation errors.

#### ༶ `validationErrors()`

**Return Value:** [InputValidationError](/api/designer-decisions/Inputs/Types/InputValidationError)[] or `null` if there were validation errors.
