## DecisionProps

All components that render a decision, such as the [&lt;ShowDecisionCard/&gt;](/components/Astro/ShowDecisionCard/) component, **require** the following props.

### ༶ `store`: [StaticDecisionStore](/api/designer-functions/Types/StaticDecisionStore)

The `store` prop provides access to all decision input data so that components can retrieve decision objects and resolve values.

The simplest way to create a decision store is, as described in the [Rendering in Astro MDX](/guides/documenting-design-decisions-with-astro-and-mdx) guide, to create a decision loader and then retrieve the store in your pages.

```mdx
import { loader } from '../decisions';

export const store = await loader();

<ShowDecision store={store} ... />
```

### ༶ `d`: string

A `string` with the name of the decision to visualize.

```mdx
<ShowDecision store={store} d="Brand Pink" size="m" />
```

<ShowDecision store={store} d="Brand Pink" size="m" />

If the decision does not exist, it renders a notice.

```mdx
<ShowDecision store={store} d="Brand Green" size="m" />
```

<ShowDecision store={store} d="Brand Green" size="m" />

## ValueProps

All components that directly render a decision or a decision value also accept the following optional props.

### ༶ `value` (true): boolean | string | object

Provide this flag to customize how the value is formatted or to hide it.

Refer to the table on [Composable Types](/components/Astro/Composable/Types) to learn which `value` options are available for each decision type.

## VizProps

All components that directly render a decision or a decision visualization also accept the following optional props.

### ༶ `size` (auto): [ShowVizSize](/api/designer-shows/Types/ShowVizSize/)

Constrains the visualization to a specific size.

```mdx
<ShowDecision store={store} d="Action Color" size={'s'} />
<ShowDecision store={store} d="Sizing Space Scale" size={'xs'} />
```

<LayoutGrid>
    <ShowDecision store={store} d="Action Color" size={'s'} />
    <ShowDecision store={store} d="Sizing Space Scale" size={'xs'} />
</LayoutGrid>

Refer to [ShowVizSize](/api/designer-shows/Types/ShowVizSize) for available sizes.

### ༶ `viz` (true): boolean | string

Provide this flag to chose between available visualizations or to hide the decision visualization altogether, displaying only its value(s).

Refer to the table on [Composable Types](/components/Astro/Composable/Types) to learn which `value` options are available for each decision type.

### ༶ `options` (optional): object

Provide this prop to configure visualization options.

Refer to the table on [Composable Types](/components/Astro/Composable/Types) to learn which `value` options are available for each decision type.
