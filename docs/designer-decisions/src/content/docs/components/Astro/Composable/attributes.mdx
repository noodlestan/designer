These components render decision details and metadata.

<Aside type="tip">
    You don't have to use these components directly in your docs. Using
    [ShowDecision](/components/Astro/ShowDecision/) will select the appropriate one for you.
</Aside>

They are exposed so that anyone can
[create custom visualizations](/guides/creating-custom-design-decision-visualizations/),
composing these directly in their custom components.

## Common Props

All of these components share the same [common props](/components/Astro/Props/#decisionprops) –
`d` and `store` – to identify and retrieve the decision.

## Decision Data

### ShowDecisionDescription

Renders the decision's description.

#### Example

```mdx

```
