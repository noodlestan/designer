## Common Props

All components that render a decision, such as the
[ShowDecisionCard](components/Astro/ShowDecisionCard/) component,
share the same common props:

-   `d` and `store`: to identify and retrieve the decision.
-   `value`, `size`, `viz`, and `options`: to control
    how the visualization and value are displayed.

Refer to [Common Props](components/Astro/Props) for more details on these props.
