export const store = await decisionLoader();

export const colorValue = createColorValue(context, '#ff0000');
export const spaceValue = createSpaceValue(context, 12);

These components render a decision's visualization and value directly.

Tip: ðŸš§ You don't have to use these components directly in your docs.
Using [ShowDecision](/components/Astro/Decisions/show-decision/) will select
the appropriate one for you.

They are exposed so that anyone can
[create custom visualizations](/guides/creating-custom-design-decision-visualizations/),
composing values and visualizations directly in their custom components.

In the [ShowDecision](/components/Astro/Decisions/show-decision) reference,
you will find all decision types and their corresponding value components.

## Common Props

All of these components share the same [common props](/components/Astro/Props/#valueprops) â€“
`value`, `size`, `viz`, and `options` â€“ to control how the visualization
and value are displayed.

Decision components, such as `<ShowDecision/>` expose these props as well, so that you
can have full control of the options.
