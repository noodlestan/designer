### Open and extensible

Open and extensible by design, this system tries to be as un-opinionated as possible about workflows and use cases.

Whether you are capturing decisions directly in JSON files or importing them from other systems, `designer-decisions` provides a set of schema-based tools for validating, transforming, and visualizing decisions.
