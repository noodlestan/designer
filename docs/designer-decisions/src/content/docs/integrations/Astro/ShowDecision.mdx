Renders the appropriate visualization according to the decision type.

In the following two examples we use the `<ShowDecision/>` to render both a [Color Value](/models/decision-types/color-value) decision and a [Space Scale](/models/decision-types/space-scale) decision.

```mdx
<ShowDecision d="Action Color" size={'s'} viz="bg" />
<ShowDecision d="Sizing Space Scale" size={'xs'} />
```

<ShowDecision d="Action Color" size={'s'} viz="bg" />
<ShowDecision d="Sizing Space Scale" size={'xs'} />

## Composes

This component composes a different value and visualization component depending on the decision type.

Refer to the table on [Composable Types](/integrations/Astro/Composable/Types) to learn about the options available.

## Props

### ༶ [DecisionProps](/integrations/Astro/Props): `store`, `d` and `layout`

These two **mandatory** `store` and `d` props serve to identify and retrieve the decision.

The optional `layout` prop controls the flow of items in set/scale decisions and the relative position of the value/viz pair.

### ༶ [ValueProps](/integrations/Astro/Props#valueprops): `value`

This prop control whether the value should be shown and how to format it.

Refer to the table on [Composable Types](/integrations/Astro/Composable/Types) to learn which `value` options are available for each decision type.

### ༶ [VizProps](/integrations/Astro/Props#vizprops): `viz`, `size`, and `options`

These props control whether the visualization should be shown, which specific visualization to show, and how to customize it.

In the examples below we are rendering a [ColorValueDecision](/models/decision-types/color-value) with, hiding the value and increasing the size of the color swatch.

```mdx
<ShowDecision d="Content Background Default Color" />
<ShowDecision d="Content Foreground Default Color" />
```

<LayoutGrid>

    <ShowDecision d="Content Foreground Default Color" viz="fg" options={{ mode: 'text' }} />
</LayoutGrid>

Refer to the table on [Composable Types](/integrations/Astro/Composable/Types) for details on available visualizations and options.
