import { Aside } from '@astrojs/starlight/components';
import {
    createColorValue,
    createOklabHueValue,
    createValueContext,
    createSpaceValue,
} from '@noodlestan/designer-decisions';
import {
    ShowColorChannelValue,
    ShowColorValue,
    ShowSpaceValue,
} from '@noodlestan/designer-shows/astro';

import { loader } from '../../../../../decisions';

export const store = await loader();

export const colorValue = createColorValue(store.createValueContext(), '#ff0000');
export const colorChannelValue = createOklabHueValue(store.createValueContext(), 123);
export const spaceValue = createSpaceValue(store.createValueContext(), 12);

These components render a decision's value directly.

<Aside type="tip">
    You don't have to use these components directly in your docs. Using
    [ShowDecision](/integrations/Astro/ShowDecision/) will select the appropriate one for you.
</Aside>

They are exposed so that anyone can [create custom visualizations](/guides/generating-documentation/creating-custom-visualizations/), composing values and visualizations directly in their custom components.

Refer to the table on [Composable Types](/integrations/Astro/Composable/Types) for all decision types and their corresponding value components.

## Common Props

### ༶ `v`: BaseValue&lt;unknown&gt;

A decision value of the appropriate type.

### ༶ `value`: string | string[]

All of these components share the same [value prop](/integrations/Astro/Props/#valueprops) to control how the value should be formatted. The formats available depend on type of value are listed below for each component.

Decision components, such as `<ShowDecision/>` expose this prop as well, so that you can have full control over format.

## Color

Color related components.

### ShowColorValue

#### ༶ `v`: [ColorValue](/api/designer-decisions/Values/Types/ColorValue)

#### ༶ `value`: [ColorFormat](/api/designer-decisions/Primitives/Types/ColorFormat) | [ColorFormat](/api/designer-decisions/Primitives/Types/ColorFormat)[]

#### Example

```mdx
export const colorValue = createColorValue(context, '#ff0000');

<ShowColorValue v={colorValue} value={['rgb', 'hsl', 'oklch']} />
```

<ShowColorValue v={colorValue} value={['rgb', 'hsl', 'oklch']} />

### ShowColorChannelValue

#### ༶ `v`: [ColorChannelValue](/api/designer-decisions/Values/Types/ColorChannelValue)

#### ༶ `value`: -

This prop has no effect on this component.

#### Example

```mdx
export const colorChannelValue = createOklabHueValue(context, 123);

<ShowColorChannelValue v={colorChannelValue} />
```

<ShowColorChannelValue v={colorChannelValue} />

## Space

<ShowSpaceValue v={spaceValue} />
